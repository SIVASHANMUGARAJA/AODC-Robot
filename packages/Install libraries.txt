Before uploading the code you have to install the necessary library

AFMotor Library 
https://learn.adafruit.com/adafruit-motor-shield/library-install

NewPing Library 
https://github.com/livetronic/Arduino-NewPing

Servo Library 
https://github.com/arduino-libraries/Servo.git //

To Install the libraries go to sketch 
>> Select the tools in Arduino Ide
>> Select Import libraries do search the libraries mentioned below;
	# AF motor v2
	# Newping
	# Servo.git
